# Machine Learning Assignments Repository

This repository contains a collection of practical machine learning assignments implemented using Python. Each assignment focuses on core concepts such as data preprocessing, visualization, regression, classification, and clustering using real-world datasets.

---

## 📁 Assignments Overview

### 🔹 Assignment 1: Data Exploration & Preprocessing

* Read data from multiple formats (CSV, XLS)
* Identify shape, missing values, data types
* Detect zero values
* Perform indexing, sorting, and selection
* Analyze attributes and unique values
* Convert data types

---

### 🔹 Assignment 2: Statistical Analysis & Data Processing

* Compute summary statistics (mean, std, variance, percentiles)
* Visualize distributions using histograms
* Perform:

  * Data Cleaning
  * Data Integration
  * Data Transformation
* Build basic classification model

---

### 🔹 Assignment 3: Data Visualization

* Scatter Plot
* Bar Chart
* Box Plot
* Pie Chart
* Line Chart
  (Using datasets from Assignment 1 & 2)

---

### 🔹 Assignment 5: Uber Fare Prediction (Regression)

* Dataset: Uber Fares (Kaggle)
* Data preprocessing & outlier detection
* Correlation analysis
* Models:

  * Linear Regression
  * Ridge Regression
  * Lasso Regression
* Evaluation Metrics:

  * R² Score
  * RMSE

---

### 🔹 Assignment 6: Diabetes Data Analysis

* Dataset: Pima Indians Diabetes Dataset
* Univariate Analysis:

  * Mean, Median, Mode, Variance, Std Dev
  * Skewness & Kurtosis
* Bivariate Analysis:

  * Linear & Logistic Regression
* Multiple Regression Analysis
* Comparison between datasets

---

### 🔹 Assignment 8: Customer Response Prediction

* Dataset: Cosmetics Shop Data
* Classification model to predict response
* Evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1 Score
* Confusion Matrix generation

---

### 🔹 Assignment 9: Customer Segmentation (Clustering)

* Dataset: Mall Customers
* Data preprocessing & train-test split
* Clustering Algorithms applied:

  * K-Means
  * (Second clustering method)
* Identify profitable customer groups

---

### 🔹 Assignment 10: Temperature Prediction (Regression)

* Dataset: India Temperature Data
* Linear Regression model
* Performance Metrics:

  * MSE
  * MAE
  * R² Score
* Visualization of regression results

---

### 🔹 Assignment 11: Admission Prediction (Classification)

* Dataset: Graduate Admissions
* Decision Tree Classifier
* Features:

  * GRE, TOEFL, CGPA, SOP, LOR, etc.
* Data preprocessing & encoding
* Model evaluation

---

### 🔹 Assignment 13: Iris Clustering

* Dataset: Iris Dataset
* K-Means Clustering
* Elbow Method to determine optimal clusters
